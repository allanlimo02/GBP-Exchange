## GBP  vs USD Currency Converter. 
### By Allan Limo

 This is a currency converter app build with springboot for  **Cellulant** graduate trainee assessment 

### Technology Used
* Spring Boot
* JPA
* MYSQL
* JWT
* Maven

### Micro-services
* User management
* JWT Authentication
